# kolo_ai_agent

Thin chat client and autonomous AI agent with local tools, memory, provider
configuration, and on-device model bootstrap support.

## Testing

Run the test suite without allowing Flutter to fetch packages implicitly:

```sh
flutter test --no-pub
```

Run coverage and enforce the current baseline:

```sh
flutter test --coverage --no-pub
dart tool/coverage_gate.dart --min-line=14.3
```

The coverage gate is intentionally dependency-free and reads
`coverage/lcov.info` generated by Flutter. The initial baseline is 14.3% line
coverage so new work can improve coverage without accepting regressions.
